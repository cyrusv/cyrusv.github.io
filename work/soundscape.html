Soundscape is a browser-based visualizer. Originally, the intention was to explore beat-detection and produce colorful visuals that were heavily tied to the beat. In order to be cross-platform, I went with a client-side javascript application, inspired by Michael Broley's project . The specific departure was to create the visuals using Processing.js, my first project in Java (not technically Java, but close).
